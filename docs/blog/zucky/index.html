最終確認です!!もう復習するのはやめにしませんか？

今回はあなたに「生で音を聴きにいくこと」に関してお話をさせていただきます

November 29, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

 

 


元ハズし王＠Zuckyです

今回は

「腹式呼吸」
についてお話しさせていただきます。

 

 

 

 

 


散々色々なことを書いてきましたが、
今更ながら腹式呼吸について書きます。


なぜなら、

 

 

 

 

 

 

 

 

腹式呼吸が出来ない者には

 

 

 

 

楽器を吹く未来がないからです。
 

 

 

 


詳しくいうと、


腹式呼吸がうまく出来ていないと


・ハイトーンがうまく出ない
・息が続かない
・無駄な力が体に入って音が硬くなる
・最悪、肺に穴が空いて楽器が吹けない


などなど、
数え始めたらきりがないほど

良くないことが起こってしまいます。



体に取り込む息は
音の素材です。

 

 

 

 

 

 


どんなに腕の良い料理人が
質の悪い食材で調理をしても

美味しい料理が出来上がらないのと
同じで

 

 

 

 

 

 

 

 

 

音も良いブレスができていないと

 

 

 

 

 

響きのある豊かな音は

 

出すことができません。

 

 


なので、しっかり理解してくださいね。



最近、僕のもとに

「腹式呼吸がうまくできているのか
　わかりません。」

との声が届いたので

今回の記事では、
腹式呼吸がどのようなものか
ということを確認する方法を

あなただけに伝授したいと
思います。


「私はもう聞かなくても大丈夫。」


という方も
これが最終確認になるように

ぜひ試してみてください！

 

 

 

 

 

 

 


では詳しい方法について
お伝えしますね！



今回お伝えする方法は

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

「こんにゃくチェック法」

 

 

 

 

 

 

 

 

 

 

「ごろんとチェック法」

 

 

 


なんと2つです !!



このどちらか好きな方を
やることで

あなたは正しい腹式呼吸を
確認できます！




ー「こんにゃくチェック法」と
　「ゴロンとチェック法」のやり方ーーーー



①「こんにゃくチェック法」



1.　足を肩幅に広げて立ちます。

2.　体を前に倒し、
　　前後屈の姿勢になります。
　　
3.　体をダラーんっとさせて
　　こんにゃくのように
　　脱力をします。

4.　息を吸ってみてください。
　　　　　　　　　　　　お腹が動くのを感じ取れますよ。




②「ゴロンとチェック法」



1.　まず仰向けになります。

2.　おへそあたりに手を
　　当ててください。

3.　息を吸ってみてください。
       　　　　　　　　お腹が動くのを感じ取れますよ。




以上になります。


この①か②を試すだけで
あなたは腹式呼吸の正しいやり方を
理解することができ、


今よりも
もっと豊かで響のある音を
出すことができるのです。


ぜひ確認してくださいね！

 

 

 

 

 

 

 

 

 

では今すぐにできることを

 

 

 

 

 

やってみましょう。

 


今やっていただきたいことは、



⑴今回学んだ①と②のどちらを
　使うのか。

⑵それをいつやりますか？

 

 

 

 

 

 


この2点を明確にして、
すぐに行動できるようにしておきましょう！


後回しにすると、
この記事の存在自体を忘れてしまい


あなたが響のある豊かな音を
出せるようになる道のりが

遠ざかってしまいますので


しっかりと
いつ、何をやるのかを明確にして

どんどん上達してしまいましょう！





いつも最後まで読んでいただいて
ありがとうございます！
