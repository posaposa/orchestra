これができたら悩みの大半は減ります！毎日ハイトーンと向き合うためのとっておきのルーティーン

August 16, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王＠Zuckyです。

 

 

今回は

 

 

とっておきの『練習法』を

 

お伝えします。

 

それは、

本番が来月に迫ったにもかかわらず

 

楽譜に記されている高い音が出ないことに

困りに困ったときに僕が知った

 

『練習法』で、

 

 

これをやったおかげで高い音が

明らかに出やすくなったと言えるほど

 

 

強力で重要な『練習法』です。

 

 

 

 

 

 

多分、あなたはこれを初めて聞くと

思います。

 

初めて聞いたあなたは、

 

 

「なにそれー」

「え、それ本当に効くの？」

「いや、そっちかー」

 

 

 

そんな声が聞こえてくるぐらい

意外なやり方だと思います。

 

 

 

 

僕は今、

ハイトーンを綺麗にかつ安定的に出す

 

ということに重点を置いて情報を

お伝えしていますが、

 

 

この『練習法』は

 

 

 

 

小さい音を安定して

吹けるようになる

効果も兼ね備えています。

 

 

 

 

 

ですので、

いち早くこの練習法を試して

 

 

あなたのチカラにしてくださいね。

 

今回お伝えするノウハウは

 

 

 

 

 

 

 

 

 

 

 

『ベリーp(ピアノ)ロングトーン』

です。

 

 

普段、ロングトーンと聞くと

 

「mfぐらいの音量で8拍吹いて4拍休む」

 

というやり方がよく聞かれると思います。

 

 

 

 

 

しかし、

この『ベリーpロングトーン』は

普通のロングトーンとは

 

一味違うんです。

 

 

 

 

ではやり方をお伝えしていきますね！

この練習方法では楽器とメトロノームを

使います。

 

 

 

ーー『pベリーロングトーン』のやり方ーーーー

 

まず基本ルールが以下の4つです。

 

・音量は全てpで行う。

 

・テンポは全て四分音符＝60

 

・スライドを1→7まで使いながら

　6つの倍音列を網羅する。

   (ex.チューニングのB♭で出る

   スライド番号1-7の音の列)

 

・音の長さは16拍伸ばして、4拍休む

 

 

 

①まずチューニングB♭の下のB♭から始めます。

　テンポに合わせて16拍ずつ半音階で

   B♭→A→A♭→G→G♭→F→E

  (スライド番号：1→2→3→4→5→6→7)

 

②①が終わったら

　今度はその上のFの倍音列で行います。

　F→E→E♭→D→ D♭→C→B

  (スライド番号：1→2→3→4→5→６→7)

 

③②が終わったら

　次の倍音列(チューニングB♭)に行きます。

　というようにこれをHighB♭まで行います。

 

 

 

どうでしょうか？

多分、あなたは「うわ大変だな」と感じたと思います。

 

 

 

 

 

それが普通です。

でも同時にワクワク感も

感じたのではないでしょうか？

 

 

 

自分がレベルアップした姿を想像すると

ワクワクしますよね？

 

 

 

ぜひトライしてみてください。

 

 

最初はできなくて当たり前です。

 

 

僕なんてチューニングB♭の上のDの倍音列で

バテて音が出ませんでした。

 

 

 

一緒に頑張って行きましょう！

 

 

 

 

この練習法をやりきったあなたは

昨日より今日、今日より明日

 

一つレベルアップしたあなた

が待っています。

 

 

 

 

具体的に言うと、

これを実施することで

あなたは背伸び音に対する潜在的なハードルに

毎日ぶち当たるので、

 

 

「なーんだ

　いつも練習してるあの音じゃん」

 

そう思えるようになり、

 

 

ラクに高い音を

出せるようになります。

 

 

 

 

では今できることを実際にやってみましょう！

 

①チューニングB♭のスライドの位置と音を

　イメージしてみましょう。

 

②そこから半音ずつ一拍毎下がって

　7番ポジションのEの音まで

　イメージしてみてください。

 

いかがだったでしょうか？

「できそう」って思えたんじゃないでしょうか？

 

 

 

 

 

この強力な練習法を知ったあなたが

背伸び音をラクに出す日までの

カウントダウンは始まっています。

 

 

 

最後までお読みいただきありがとうございます。

部活の先輩に聞くノリで

楽器に関する相談を

 

"期間限定"で

 

受け付けています！
