結局どっち、ムネなの？ハラなの？

August 12, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王@Zuckyです。

 

前回は

『ブレス』について

お話しさせていただきました。

 

 

ですので今回はブレスに関係の深い

『腹式呼吸』についてお話しして

いきたいと思います。

 

 

 
 
この文章を最後まで読み終えたとき
 
あなたは
普段から胸や肩で息を吸うことが
できないぐらい
 
空気をお腹で吸うことを
意識するようになり、
 
ハイトーンを軽々しく扱うことの
てきるトロンボーンプレーヤーになります。
 
 
 
また、
長時間楽器が吹き続けられる体になりますので
結果的に他の人よりも長く
楽器を楽しめることになります。
 
 
 
逆に、
このノウハウを知らないままあなたが練習を
続けていくと、体を壊しかねません。
 
 
 
特に高音域を安定的に吹けるようにしようと
努力をしているあなたは、
肺の圧力が普通の音に比べて高くなるので、
 
 
最悪の場合、
肺に穴が空いて、その穴がふさがるまでの
 
数ヶ月間完全に
楽器を吹けなくなってしまいます。
 
 
今のあなたの生活から楽器を取ってしまうと、
どのようなことが起きるでしょうか？
 
 
 
 
「コンクールに出れない」
 
「オーディションを
    受けれない」
 
「後輩に抜かされてしまう」
 
 
 
そんな恐怖が頭をよぎるのでは
ないでしょうか？
 
 
 
 
「一日楽器を吹かないと、
    元の状態に戻すのに3日かかる」
 
と言われてますので、
あなたが肺に穴を空けたその瞬間、
 
 
 
約半年ものブランクを
背負うこととなるのです。
 
 
 
これでは
あなたが高い音を響かせる日がどんどん
遠ざかっていってしまいます。
 
 
 
絶対にそんなことは
避けたいですよね？
 
 
 
僕も高音が出なくて
先生に怒られ続けた地獄の日々を経験している身なので、
 
 
一日でも早く、
あなたに「高音を安定的に鳴らせる」ように
してもらいたいです！
 
 
なので、
これからお伝えする方法を
今すぐにでも自分のものとして活用して下さい！
 
 
では『腹式呼吸』について
一緒に考えていきましょう！
 
 
まず『腹式呼吸』が普通の呼吸とどう違うか
ということについてお話しします。
 
一番の大きな違いは、呼吸の深さです。
 
 
『腹式呼吸』はみぞおち辺りにある
横隔膜という体の一部を下に下げることで
肺に空気を入れます。
 
 
これによって、肋骨を動かして呼吸をする
胸式呼吸よりも深く空気を吸うことができます。
 
 
 
あなたも「お腹で吸って」と言われた
経験があるかもしれませんが、あれ、
 
 
 
実際にお腹に空気が入っている
わけではないんです！！！
 
 
 
 
正しくは肺に空気が入る
と横隔膜が下がるので
 
 
お腹に空気が入っているように思えるんです。
 
 
この正しい『腹式呼吸』をマスターして頂くために
今回お伝えするテクニックが
 
 
 
 
 
 
 
 
 
 
『タピオカストロー理論』です。
 
 
『腹式呼吸』を体感して頂く方法はいくつか
あるのですが、その中でも、どこでもできるのが
 
 
この『タピオカストロー理論』なんです。 
 
 
 
あなたはタピオカジュースを
飲んだことがありますか？
 
 
実はあのジュースを飲むときのタピオカストローと、
ストローの直径より少し大きなタピオカを吸うときの
口の形が『腹式呼吸』を体感するにはピッタリなんです。
 
 
 
 
ー『タピオカストロー理論』のやり方ーーーーーー
 
まず姿勢を正してから①から③を試してみてください。
 
①タピオカジュースを飲むことをイメージする。
 
②ストローの直径よりも3mm大きなタピオカを
　今から吸い上げるイメージをする。
 
③②のタピオカを食べるため、一度息を吐ききってから
　一気に吸い上げる。
 
 
どうだったでしょうか？
 
『タピオカジュース理論』を実践したときに
お腹を使って呼吸ができているはずです。
 
 
これが高音を安定して鳴らすために必要な
『腹式呼吸』です。
 
ぜひあなたがトロンボーンを吹いていて
「ちゃんと腹式呼吸で空気吸えてるかな？」と
不安が頭をよぎったときには、
 
この『タピオカジュース理論』を思い出して
確認してください。
 
 
 
では今すぐにできることを実際にやってみましょう‼︎
 
①今目の前にタピオカジュースがあるとイメージします。
 
②そのタピオカストローをくわえてみて下さい
 
 
いかがだったでしょうか？
 
その口の形で息を吸うことで
『腹式呼吸』を体感できると思います。
 
 
そして、
しっかりと『腹式呼吸』を使ってトロンボーンを
吹く練習を重ねることで安定した高音が
鳴らせるようになります！
 
 
 
最後まで読んでいただきありがとうございます。
