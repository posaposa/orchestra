ビート板を使った経験がある方が管楽器を吹くのに有利なたった一つの理由。

August 13, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王@Zuckyです。

 

 

今回は

『テクニックの関連性』

について

 

お腹の使い方と腹式呼吸を例に

お話させていただきます。

 

これまで高い音を

 

綺麗に

そして、

安定的に出すことについて

 

お話しさせていただきました。

 

ただ今までの一つ一つの要素を

独立したものとして

捉えてしまうと

 

 

 

実は上手く高音を出すことには

繋がらないんです。

 

 

 

 

 

というか今のあなたの

実力レベルから

 

一つも二つも

ランクダウンしてしまう

可能性があることも

 

 

否定できません。

 

 

そうしますと、

上手くできないイライラから

 

スランプに陥ったと勘違いをして、

 

練習するのがどんどん苦痛に

なっていきます。

 

 

 

 

そんな状態でトロンボーンが

上手くなりますか？

 

 

 

 

 

少なくとも、

僕は上手くなるとは思えません。

 

暗い気持ちで楽器を吹いていても

ただ鉄の塊に空気を流し込んでいる

のと変わりはないので

 

その時間を

他の楽しいことに使った方が

良いと思います。

 

 

 

今回のお話を最後まで

お読みいただければ、

 

あなたは

一つ一つのテクニックを繋ぎ合わせる

重要性を理解することができて、

 

 

今後習得するテクニックの

定着が他の人よりも

段違いに速くなると思います。

 

 

 

では今から、

前回の『お腹の使い方』と

前々回の『腹式呼吸』を例に用いて

テクニックをお伝えしたいと

思います。

 

 

今回のテクニックは

 

 

 

 

 

 

 

 

 

 

 

『ビート板理論』です。

 

 

 

 

「ビート板？プールで使うやつ？」

 

とピンっときたあなた。

そうです。あれなんです。

 

 

あなたはプールに行ったとき

ビート板をプールの中に

沈めたことがありますか？

 

 

水面と平行に沈めて行くとき

特に大きな水の抵抗を感じますよね？

 

また

 

水中に沈めれば沈めるだけ

自分の方に跳ね返してくる

力は大きくなりますよね。

 

 

 

あのイメージがポイントなんです。

 

あれをトロンボーンを吹くということに

落とし込んでみると、

 

 

ビート板を押し込む手が、あなたの肺。

 

水と平行になったビート板が横隔膜。

 

跳ね返してくる水の役割が腹筋。

 

 

となります。

何が言いたいかといいますと、

 

ブレスを深く吸って

空気が肺にたくさん

入っていれば

 

抵抗する腹筋が

強くなる(=上手く使える)んです。

 

 

 

逆にブレスがしっかりと

とれていないと、

お腹に力が入りません。

 

 

ー『ビート板理論』のやり方ーーーーー

 

①息をこれ以上吐ききれない

　というところまで

　しっかり吐く。

 

②その反動で息をしっかり

　体に入れ込む。

 

③お腹に力を入れて

　息を吐く。

 

以上になります。

 

 

「だからなんなの？」

と思われた方もいらっしゃるかも

しれません。

 

 

 

では実際に比較して違いを

感じてみてください！

 

A.ブレスを浅くとってお腹に力を

　入れて、息を吐いてみる。

 

 

B.腹式呼吸を意識して

　ブレスを深くとり、お腹に力を

　入れて息を吐いてみる。

 

 

 

いかがだったでしょうか？

 

あなたはAとBのどちらの方が

高音を吹く上で必要なお腹の力や

息のスピードだと思いましたか？

 

差は明らかだったのでは

ないでしょうか？

 

この

 

「ガッシリとしたお腹の力」

や

「息のスピード感」が

 

 

高音を安定的に出すために

必要となるんです。

 

 

あなたはもうお分かりのように、

 

一つ一つのテクニックを

独立的に活用させるのではなく、

 

繋ぎ合わせることで

可能となったことですよね？

 

今回のように一つ一つのテクニックの

関連性を意識することで高音を出しやすく

することは山ほどあるので、

 

今後も

テクニックとテクニックの

 

繋がり

を意識して

 

効率良くメキメキとトロンボーンを

上達させてくださいね。

 

 

 

最後までお読みいただきありがとうございます。
