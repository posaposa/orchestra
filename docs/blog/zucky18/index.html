部分的ガチガチマッチョになろう！

August 14, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王@Zuckyです。

 

 

 

今回は

ちょっと背伸びをした音(いわゆる高音)を

出すために必要な

 

 

『口周りの筋肉』

 

 

について

お話ししていきたいと思います。

 

 

 

ここからは高音のことを

「背伸び音」と呼んでいきます。

 

 

(背伸び音を詳しく知りたい方は前回の

「特別扱いは問題が生じやすい。」を

 ご覧ください。)

 →http://ameblo.jp/posauneo/entry-12301294902.html

 

 

 

 

この口周りの筋肉を鍛えることで、

あなたの背伸び音は

 

 

 

劇的に安定して

出せるようになります。

 

 

 

これは僕の知り合いの

音大に通っているトランペットの方が

言っていたことです。

 

 

 

 

ある日その方はプロの奏者の

レッスンを受けていたそうですが、

 

そのプロの方が演奏をしているときに

口の周りを触ってごらんと言われ、

触ったり押したりしたところ

 

 

 

 

 

ビクともしない鋼のような

 

 

口周りの筋肉だったそうです。

 

 

 

やっぱり口周りがガッシリと

していないと、すぐにバテてしまうので

口周りの筋肉を鍛えることは重要ですね。

 

 

 

確かにトロンボーンを毎日練習していれば

口周りの筋肉は鍛えていかれます。

 

 

 

 

しかし、

一日でも早く、

背伸び音を安定的にかつ綺麗に出したいですよね？

 

 

プルプルと震える背伸び音を

 

 

いつまでもいつまでも

周りに聞かせたくないですよね。

 

 

探さない。待つの。

 

 

 

 

ではダメなんです。

 

 

ぜひこれをやってみてください。

お金も全然かかりません。

 

 

 

今回お伝えするのは

 

 

 

 

 

 

 

『ストロー・デ・トレーニング』

です。

 

 

 

 

 

あのプラスチックでできた

フニャフニャのストローを用いて、

 

ガチガチの筋肉を作り上げるんです。

 

もしあなたの近くにストローがあれば

一緒にやってみましょう！

 

 

 

 

ー『ストロー・デ・トレーニング』のやり方ーーーーー

 

①真っ直ぐ前を向きましょう。

 

②ストローを口の真ん中に置いて

　先っぽの約1.5cmをくわえましょう。

 

③自分の顔と平行になるように

　ストローを持ち上げます。

　

※このとき、口の両端を意識することが

　ポイントです。

 

以上です。めちゃめちゃ簡単ですよね。

 

目安として、毎日5-10分ほど意識的に

続けることが効果的です。

 

 

 

 

これならテレビや動画を見ながらでも

やることができますので、

ぜひ試してみてください。

 

では、

今すぐできることを

ここでやってみましょう。

 

 

①あなたがトロンボーンを吹くときの

　口の形を思い浮かべて

　実際に作ってみてください。

 

②その口の筋肉がガチガチになっている

　想像をしてください。

 

これでイメージトレーニングは

カンペキです。

 

 

 

 

今回の『ストロー・デ・トレーニング』を

使って、鋼のような口周りを作ってください。

 

 

最後までお読みいただき

ありがとうございます。
