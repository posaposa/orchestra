トロンボーンを吹く人は天狗にはなってはなりません。ピノキオになりなさい。

August 13, 2017

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王＠Zuckyです。

 

 

 

今回は

トロンボーンを

吹くときにおける

 

『姿勢』について

お話させていただきます。

 

 

 

今回お話しさせていただく

楽器を吹くときの姿勢を

マスターすると、

 

あなたのベルから出た

ハイトーンは

 

のびのびと

聴いている人の方へと

飛んでいき、

 

 

「あの素晴らしい音色は

　なんだったんだろう？」

 

 

と聴いている人の

記憶に残るような音を

出すことできるようになります。

 

 

 

 

逆に

正しい『姿勢』を習得することができないと、

 

あなたの音は2m先の

木管楽器たちを越えられず

 

 

聴衆の耳に届くことは

一生ありません。

 

 

それだけ

楽器を吹く『姿勢』は

 

高音を吹く上で

重要な鍵を握っているんです。

 

トロンボーンを構える姿勢は

ぶっちゃけ十人十色ですよね。

 

 

 

けれどトロンボーンを吹く

 

 

『胴体の姿勢』に差異は

ほとんどありません。

 

 

 

 

 

 

 

試しにあなたの思い浮かぶ

プロのオーケストラや

吹奏楽団を想像してみてください。

 

 

 

 

 

 

猫背で吹いているプレーヤーは

いましたか？

 

僕はNHK交響楽団を想像しましたが

少なくとも僕の記憶上で

 

 

背中が丸まっているプレーヤーは

誰一人としていません。

 

 

 

僕も高い音が出なくて

悩み続けていたときは、
色々な楽器の構え方や角度など
様々なことを試しましたが
 
結局ここにたどり着きました。
それが今回お伝えする
 
 
 
 
 
 
 
 
 
 
『操り人形理論』です。
 
 
この理論は操り人形のように
 
「自分が上から糸で吊るされている
　ような感覚」
 
を持つことで、
 
トロンボーンを吹く
 
「最適な姿勢」
 
を習得することができます。
 
 
 
この理論を実行すると

3つの良いことがあります。

 

まず、上から糸で吊るされている意識を

持つことで、背筋が伸びて

 

「お腹の筋肉が使いやすくなる。」

 

次に、背筋が伸びると胸が少し張るので

肺の周りが広くなり、

 

「肺に空気をとりこみやすくなる」

のと同時に

「とり入れられる量も増える。」

 

 

 

 

そして最後に、変なところに力が入らず

 

「リラックスして

　トロンボーンを吹くことができる。」

 

 

以上のように悪いことは探しても

見つからないぐらい

 

重要なノウハウです。

 

 

 

ー『操り人形理論』のやり方ーーーーー

 

①上から糸で吊るされている感覚を持つ。

 

②腹式呼吸でブレスをして、

　少しお腹に力を入れながら息を吐いてみる。

 

 

以上になります。

まずメリットはたくさんあるのに

 

めちゃめちゃ簡単ですよね！

 

 

 

 

では今すぐにできることを

実際にやってみましょう！

 

①操り人形のになります。

 

②あなたを操っている人が

　あなたの頭についている紐を

　持ち上げたことを

　想像してください。

 

いかがだったでしょうか？

 

自然と背筋が伸びて、

体がまっすぐになったと

思います。

 

 

この『操り人形理論』を使うことで

あなたの楽器を吹く姿勢は

リラックスした状態になり、

 

今までのテクニックを

より効果的に使うことができます。

 

 

 

 

そうしたときには、

あなたは聴いている人を一人残らず魅了する

 

輝かしくて力強いハイトーン

 

を響かせることが

できます。

 

 

 

最後までお読みいただき

ありがとうございます。

 

 

LINE＠では

部活の先輩に聞くノリで

楽器に関する相談を

 

“期間限定”で

 

受け付けています！

LINE@でZuckyを活用する
 

＜LINE＠追加の仕方＞

 

・スマートフォンの場合

 

①上のボタンをクリックすると

　LINEアプリが起動します。

 

②ログイン後

「ZUCKY@

  TROMBONE CONSULTANT」

　と表示されるので

 

　「追加」を押します。

 

③トーク画面にて

　質問や聞きたいこと

　相談などを送信してください。
