上手くなりたければ「ブーブー」言おう。

August 15, 2017

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

 

元ハズし王＠Zuckyです。

 

 

今回はトロンボーンでそもそも

音を出すために必要な

 

 

『バズィング』

 

 

についてお話ししていきたいと思います。

 

『バズィング』は口やマウスピースだけでやる

あの「ブー」ってやつです。

 

 

『バズィング』はトロンボーンで

音が出る仕組みと直結しているので、

 

 

どんな高さの音が出るか

とか

どんな音色で音が出るか

 

というその奏者の力量をもろに

浮き彫りにさせる要素の一つと言えます。

 

 

 

 

ですので、あなたが今回の内容を

最後まで読み終わったときには、

 

 

あなたの『バズィング』に対する思いや

こだわり方が読む前と自分でも

 

見違えるほど変わっていると

思います。

 

逆にしっかりと唇を振動させることが

できていなければ、

 

 

絶対に高い音を

響かせることはできません。

 

 

 

 

もし十分な振動がないまま

背伸び音が出たとしても、

 

それは力ずくで出した音なので、

 

 

実際に使うことはできないです。

 

 

 

しかし今回お伝えするテクニックを

あなたが実践することで、

 

 

「背伸び音」五線の中にある音と

遜色変わりなく、ラク〜に

トロンボーンのベルから出すことが

できるようになるのです。

 

 

 

 

まず今回は

自分が『バズィング』をちゃんと

できているのかどうか、

一緒に確認してみましょう！

 

 

今回お伝えするテクニックは

『ウィーン作戦』です。

 

 

 

 

すみません。

オーストリアの首都ではありません。

良い名前が思いつかず、

効果音にしてしまいました。

 

 

この『ウィーン作戦』を実行することにより、

あなたの唇が十分に振動しているかどうかを

確認することができます。

 

 

今回は楽器を使って行います。

ではやり方を説明していきますね。

 

 

 

ー『ウィーン作戦』のやり方ーーーーーーーー

 

①まず楽器構える

 

②チューニングのB♭を伸ばしてください。

 

③B♭を出したまま、構えている楽器を

　スライドの角度と平行に、

　ウィーンと前に出して

　口からマウスピースを外してみてください。

 

④このとき、唇だけで『バズィング』が

　できているか確認してください。

 

いかがだったでしょうか？

 

多分、『バズィング』は楽器から口を

外した途端に終わってしまって

続いていないと思います。

 

 

 

僕もそうでした。

最初からこれができている人は、

ほとんどいません。

 

 

すでにできていたら、

背伸び音で困っていないかも

しれませんね。

 

でも、できなければ

 

練習してできるように

すればいいんです。

 

 

 

 

その練習については

 

次回のPart2で

お話ししていきたいと思います。

 

では今すぐにできることを

やってみましょう。

 

①マウスピースを鳴らすときの

　口を作る。

 

②バズィングはしなくて良いので、

　細い一本の糸のような息を意識して

　実際に息を吐いてみる。

 

これに唇が振動すれば

背伸び音が出るようになります。

 

まず、

あなたのバズィングを

『ウィーン作戦』を用いて確認してみてくださいね。

 

 

 

最後までお読みいただき

ありがとうございます。
